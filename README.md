## 🚀 Project Overview
Built an end-to-end customer analytics project simulating a corporate-grade data workflow. The objective was to transform raw transactional data into strategic business intelligence using Python, SQL, Excel, and Power BI.

---

## 🎯 Objectives
- Perform data cleaning and transformation  
- Conduct exploratory data analysis (EDA)  
- Analyze customer segments and purchasing behavior  
- Identify loyalty patterns and revenue drivers  
- Deliver actionable business recommendations  

---

## 🛠️ Tech Stack
- **Python (Jupyter Notebook)** – Data cleaning & EDA  
- **SQL** – Business transaction simulation & analytical queries  
- **Excel** – Data preprocessing & validation  
- **Power BI** – Interactive dashboards & KPI visualization  

---

## 🔄 Project Workflow

### 1️⃣ Data Preparation & EDA (Python)
- Handled missing values and inconsistencies  
- Transformed raw dataset for structured analysis  
- Performed exploratory data analysis to identify trends  

### 2️⃣ Data Analysis (SQL)
- Analyzed customer segmentation  
- Evaluated repeat purchase behavior  
- Measured revenue contribution  
- Identified loyalty patterns  

### 3️⃣ Visualization & Insights (Power BI)
Developed an interactive dashboard highlighting:
- Key KPIs  
- Sales & revenue trends  
- High-value customer segments  
- Retention insights  

### 4️⃣ Report & Presentation
- Documented key findings  
- Provided strategic business recommendations  
- Created a stakeholder-ready presentation  

---

## 📊 Key Insights
- Identified high-value and repeat customer segments  
- Analyzed purchasing trends and revenue drivers  
- Delivered data-driven recommendations to improve retention and growth
